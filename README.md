# LibraryMS
winform C# 图书馆管理系统
## 开发环境
1. 系统开发平台：Microsoft Visual Studio 2019。
2. 系统开发语言：C#。
3. 数据库管理软件：Microsoft SQL Server 2008R2。
4. 运行平台：Windows 10。
5. 运行环境：Microsoft .NET Framework SDK v4.7
6. 控件库： [SunnyUI](https://github.com/yhuse/SunnyUI)
